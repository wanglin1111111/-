# 引用格式规范

## 一、APA格式（美国心理学会）

### 1. 文内引用

**单一作者：**
```
研究表明（Smith, 2020），...
Smith（2020）发现...
```

**两位作者：**
```
研究表明（Smith & Jones, 2019），...
Smith和Jones（2019）发现...
```

**三位及以上作者：**
```
研究表明（Smith et al., 2018），...
```

**直接引用：**
```
Smith（2020）指出："..."（p. 15）
```

### 2. 参考文献列表

**书籍：**
```
Smith, J. (2020). Title of book (3rd ed.). Publisher.
```

**期刊文章：**
```
Smith, J., & Jones, M. (2019). Title of article. Journal Name, 15(2), 100-120. https://doi.org/xxxxx
```

**网页：**
```
Smith, J. (2020, January 15). Title of webpage. Website Name. URL
```

## 二、MLA格式（现代语言协会）

### 1. 文内引用

**单一作者：**
```
研究表明（Smith 45），...
Smith指出...（45）
```

**两位作者：**
```
研究表明（Smith and Jones 23），...
```

**三位及以上作者：**
```
研究表明（Smith et al. 56），...
```

### 2. 参考文献列表

**书籍：**
```
Smith, John. Title of Book. Publisher, 2020.
```

**期刊文章：**
```
Smith, John, and Mary Jones. "Title of Article." Journal Name, vol. 15, no. 2, 2019, pp. 100-120.
```

**网页：**
```
Smith, John. "Title of Webpage." Website Name, 15 Jan. 2020, URL.
```

## 三、Chicago格式（芝加哥手册）

### 1. 脚注/尾注格式

**首次引用：**
```
1. John Smith, Title of Book (Place: Publisher, 2020), 45.
```

**再次引用：**
```
2. Smith, Title of Book, 50.
```

### 2. 参考文献列表

**书籍：**
```
Smith, John. Title of Book. Place: Publisher, 2020.
```

**期刊文章：**
```
Smith, John. "Title of Article." Journal Name 15, no. 2 (2019): 100-120.
```

## 四、GB/T 7714格式（中国国家标准）

### 1. 文内引用

**顺序编码制：**
```
研究表明[1]，...
张三等[2]发现...
```

**著者-出版年制：**
```
研究表明（张三，2020），...
张三（2020）发现...
```

### 2. 参考文献列表

**专著：**
```
[1] 张三. 书名[M]. 北京: 出版社, 2020: 45-50.
```

**期刊文章：**
```
[2] 张三, 李四. 文章标题[J]. 期刊名, 2020, 15(2): 100-120.
```

**学位论文：**
```
[3] 张三. 论文标题[D]. 北京: 北京大学, 2020.
```

**网页：**
```
[4] 张三. 网页标题[EB/OL]. (2020-01-15)[2020-05-01]. http://www.example.com
```

## 六、引用注意事项

### 1. 直接引用

- 必须加引号
- 注明页码
- 引用内容准确无误

### 2. 间接引用

- 不需要加引号
- 用自己的话转述
- 仍需注明出处

### 3. 多次引用同一文献

- 首次引用需完整信息
- 后续引用可简化

### 4. 引用网页

- 注明访问日期
- 保留URL
- 注意网页可能更新或删除

## 七、常见错误

### 1. 格式不统一

**错误：** 混用不同格式
**正确：** 全文使用统一格式

### 2. 信息不完整

**错误：** 缺少出版信息
**正确：** 提供完整的文献信息

### 3. 引用与正文不对应

**错误：** 正文引用[1]，参考文献列表没有[1]
**正确：** 确保一一对应

### 4. 二手引用

**错误：** 直接引用转引内容
**正确：** 找到原始文献，或注明"转引自"
