# 写作风格指南

## 风格类型概述

### 1. 学术风格

**特点：**
- 客观、严谨、逻辑清晰
- 使用专业术语
- 被动语态较多
- 引用规范严格

**适用场景：**
- 学术论文
- 研究报告
- 技术文档

**示例：**
```
本研究采用定量分析方法，对500名受访者进行了问卷调查。
结果表明，变量X与变量Y之间存在显著正相关关系（r=0.67, p<0.01）。
```

### 2. 商业风格

**特点：**
- 简洁、专业、数据驱动
- 重点突出
- 行动导向
- 结果导向

**适用场景：**
- 商业报告
- 项目提案
- 市场分析

**示例：**
```
Q3营收增长23%，超出预期目标5个百分点。
建议：加大市场投入，预计Q4增长30%。
```

### 3. 创意风格

**特点：**
- 生动、形象、情感丰富
- 修辞手法多样
- 个性化表达
- 注重氛围营造

**适用场景：**
- 小说创作
- 散文随笔
- 广告文案

**示例：**
```
夕阳像熔化的金子，缓缓沉入地平线。
晚风带着海水的咸味，轻抚过她疲惫的脸庞。
```

### 4. 新闻风格

**特点：**
- 客观、简洁、时效性强
- 倒金字塔结构
- 事实为主
- 引用权威

**适用场景：**
- 新闻报道
- 时事评论
- 行业资讯

**示例：**
```
据新华社报道，今日上午10时，新型运载火箭在文昌发射中心成功发射。
这是我国航天事业的又一重大突破。
```

### 5. 技术风格

**特点：**
- 准确、清晰、步骤明确
- 术语规范
- 示例充分
- 可操作性强

**适用场景：**
- API文档
- 使用手册
- 技术教程

**示例：**
```
使用GET方法请求/api/v1/users端点，返回用户列表。
参数：
- page: 页码，默认1
- limit: 每页数量，默认20，最大100
```

## 风格选择指南

| 目标读者 | 推荐风格 | 语言特点 |
|----------|----------|----------|
| 专家学者 | 学术风格 | 专业术语、严谨客观 |
| 企业管理层 | 商业风格 | 简洁高效、数据支撑 |
| 普通大众 | 通俗风格 | 通俗易懂、生动有趣 |
| 技术人员 | 技术风格 | 准确规范、步骤清晰 |
| 文学爱好者 | 创意风格 | 优美形象、情感丰富 |

## 风格一致性原则

1. **全文统一**：一篇文章应保持风格一致
2. **段落协调**：段落之间风格过渡自然
3. **句式匹配**：长句与短句搭配得当
4. **词汇统一**：专业术语使用一致
