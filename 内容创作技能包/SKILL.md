# 内容创作技能 (Content Creation Skill)

## 描述
内容创作技能是一款专业的AI辅助内容生产工具，旨在帮助用户高效产出高质量的文章、文案、脚本等各类内容。通过结构化的创作流程和智能化的辅助功能，降低创作门槛，提升内容质量，实现从灵感构思到成品输出的全流程覆盖。

## 核心功能

### 1. 文章结构规划 (`create-outline`)
- **功能说明**: 根据主题自动生成逻辑清晰的文章大纲
- **适用类型**: 公众号文章、博客文章、新闻稿、白皮书、行业报告
- **输出内容**: 
  - 文章标题建议（3-5个备选）
  - 核心论点与分论点结构
  - 段落逻辑顺序
  - 关键素材提示
- **使用示例**: 
  - `为"碳中和背景下新能源汽车发展趋势"创建文章大纲`
  - `规划一篇关于职场时间管理的深度文章结构`

### 2. 文案润色优化 (`polish-text`)
- **功能说明**: 对现有文本进行语言优化，提升表达力和感染力
- **优化维度**:
  - 语言流畅度：修正语法错误，优化句式结构
  - 表达生动性：增加修辞手法，提升画面感
  - 逻辑严密性：强化论证链条，消除逻辑漏洞
  - 情感共鸣：调整语气语调，增强读者共鸣
- **使用示例**:
  - `润色这段产品介绍，使其更具吸引力`
  - `优化这篇推广文案的转化率`

### 3. 创意点子生成 (`generate-ideas`)
- **功能说明**: 基于主题或产品快速生成创意内容方案
- **创意类型**:
  - 标题创意：吸睛标题、悬念标题、数字标题等
  - 角度创意：独特切入点、反差视角、故事化表达
  - 形式创意：图文结合、视频脚本、互动内容
  - 话题创意：热点结合、争议话题、情感共鸣点
- **使用示例**:
  - `为智能手表生成20个短视频创意标题`
  - `围绕"春季养生"主题生成10个内容角度`

### 4. 多平台适配改写 (`adapt-platform`)
- **功能说明**: 将同一内容适配为不同平台的风格要求
- **支持平台**:
  - **小红书**: 种草风格、emoji点缀、口语化表达
  - **抖音/快手**: 短视频脚本、黄金3秒开头、互动引导
  - **公众号**: 深度长文、专业论述、价值输出
  - **微博**: 短平快、话题标签、热点结合
  - **知乎**: 问答形式、专业分析、逻辑论证
  - **B站**: 年轻化表达、弹幕文化、知识分享
- **使用示例**:
  - `将这篇公众号文章改写成小红书风格`
  - `把这个产品介绍适配为抖音短视频脚本`


## 最佳实践

### 标题创作原则
1. **数字法则**: 使用具体数字增加可信度（如"5个方法"、"3天学会"）
2. **悬念设置**: 制造信息缺口激发好奇心
3. **痛点直击**: 直接点出目标受众的核心困扰
4. **利益承诺**: 明确告知读者能获得什么价值
5. **情感共鸣**: 触发读者的情绪反应

### 内容结构模板

**AIDA模型**（适用于营销文案）
- Attention（注意）: 吸引注意力的开头
- Interest（兴趣）: 激发兴趣的内容展开
- Desire（欲望）: 建立渴望的价值呈现
- Action（行动）: 明确的行动号召

**SCQA模型**（适用于深度文章）
- Situation（情境）: 背景介绍
- Complication（冲突）: 问题或挑战
- Question（疑问）: 提出的核心问题
- Answer（答案）: 解决方案或观点

**金字塔原理**（适用于专业论述）
- 结论先行
- 以上统下
- 归类分组
- 逻辑递进

| B站 | 知识分享、年轻化 | 5-15分钟脚本 | 梗文化、弹幕互动 |

## 进阶技巧

### 1. 热点借势创作
- **监测工具**: 微博热搜、百度指数、新榜
- **借势原则**: 相关性>时效性>争议性
- **创作公式**: 热点事件 + 专业视角 + 价值输出

### 2. SEO优化写作
- **关键词布局**: 标题、首段、小标题、结尾自然融入
- **长尾关键词**: 挖掘用户真实搜索意图
- **内容质量**: 原创度、信息密度、用户体验

### 3. 数据驱动优化
- **核心指标**: 阅读量、完读率、互动率、转化率
- **A/B测试**: 标题测试、封面测试、发布时间测试
- **迭代优化**: 基于数据反馈持续改进内容策略

### 4. 个人IP打造
- **定位清晰**: 垂直领域 + 独特视角 + 稳定输出
- **风格统一**: 视觉风格、语言风格、价值观一致
- **互动运营**: 评论区维护、粉丝社群、定期活动

## 常见问题解答

**Q: 如何确定内容选题？**
A: 结合热点趋势、用户痛点、专业领域三个维度，使用generate-ideas挖掘高潜力选题。

**Q: 不同平台的内容如何平衡原创与效率？**
A: 核心观点保持一致，表达方式因地制宜。使用adapt-platform功能实现"一鱼多吃"。

**Q: 如何提升文案的转化率？**
A: 关注用户决策路径，在关键节点设置行动引导。使用polish-text优化CTA（行动号召）文案。

**Q: 创作遇到瓶颈怎么办？**
A: 切换创作场景、参考竞品案例、使用generate-ideas获取灵感、暂时搁置换个思路。
