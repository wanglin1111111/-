# 内容创作技能 (Content Creation Skill)

## 描述
专业的内容创作辅助工具，帮助用户进行文章、文案、脚本等内容的创作。

## 功能
- 文章结构规划
- 文案润色优化
- 创意点子生成
- 多平台适配改写

## 使用场景
- 公众号文章撰写
- 短视频脚本创作
- 广告文案设计
- 社交媒体内容

## 工具
- `create-outline`: 创建内容大纲
- `polish-text`: 润色文本
- `generate-ideas`: 生成创意点子
- `adapt-platform`: 适配不同平台

## 示例
```
创建一个关于环保的文章大纲
润色这段产品介绍文案
为这个产品生成10个创意标题
把这篇文章改写成小红书风格
```
